Pizza Sales Analysis - SQL Project

Project Overview:

This project analyzes pizza sales data using SQL queries to gain insights into sales trends, customer preferences, and revenue generation. The dataset includes information on orders, pizza sizes, categories, and prices.

Dataset Overview :

The dataset consists of multiple tables related to pizza sales, including:

Orders: Contains order details, including timestamps.

Order Details: Links orders with specific pizzas.

Pizzas: Information on pizza types, sizes, and prices.

Pizza Categories: Categorizes pizzas into different groups.


Insights Gained:

The highest revenue-generating pizza and the most frequently ordered pizza size were identified.

Peak order hours help in understanding customer behavior and optimizing operations.

Category-wise performance analysis highlights popular pizza categories.

Revenue contribution analysis provides insights into high-value pizza types


Conclusion:

This project provides valuable insights into pizza sales, customer preferences, and revenue distribution. The findings can help businesses optimize their menu, pricing, and marketing strategies.
